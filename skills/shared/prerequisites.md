# CLI Prerequisites

Check `mthds-agent` availability before running commands:

1. Try `mthds-agent --version`
2. If not found, guide install: `npm install -g mthds`

All `mthds-agent` commands also accept a `--log-level` global option before the subcommand (e.g., `mthds-agent --log-level debug pipelex <command>`).
